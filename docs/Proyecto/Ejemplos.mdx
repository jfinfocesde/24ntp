## Curso de Pandas: Estructura Básica

Este curso te introducirá a la estructura fundamental de Pandas, una biblioteca de Python esencial para el análisis de datos. Aprenderás los conceptos básicos para manipular y analizar datos de manera eficiente.

**Módulo 1: Introducción a Pandas**

* **¿Qué es Pandas?**
    * Explorando las capacidades de Pandas para la manipulación y análisis de datos.
    * Ventajas de usar Pandas en comparación con otras herramientas.
* **Instalación de Pandas**
    * Guía paso a paso para instalar Pandas en tu entorno de Python.
* **Importando la biblioteca**
    * Aprender a importar Pandas correctamente en tus scripts de Python.
* **Series:**
    * Comprender el concepto de Series como vectores unidimensionales etiquetados.
    * Crear Series a partir de listas, diccionarios y arreglos NumPy.
    * Acceder y modificar valores en Series.
    * Operaciones básicas con Series (suma, multiplicación, etc.).
* **DataFrames:**
    * Entender DataFrames como estructuras de datos bidimensionales tabulares.
    * Creación de DataFrames a partir de listas, diccionarios, Series y archivos CSV.
    * Acceso a filas y columnas mediante índices y nombres.
    * Operaciones básicas con DataFrames (filtrado, selección, etc.).

**Ejemplo completo con todas las modificaciones:**

```python
import pandas as pd

# 1. Crear DataFrame
data = {'Nombre': ['Ana', 'Juan', 'Pedro', 'Sofía', 'Carlos', 'María', 'Luis'],
        'Edad': [18, 19, 18, 17, 20, 18, 19],
        'Género': ['Mujer', 'Hombre', 'Hombre', 'Mujer', 'Hombre', 'Mujer', 'Hombre'],
        'Calificación': [8.5, 7.0, 9.2, 9.8, 6.5, 8.0, 7.8]}
df = pd.DataFrame(data)

# 2. Agrupación por Género
grouped_gender = df.groupby('Género')

# 3. Resumen de Edad
edad_promedio = grouped_gender['Edad'].mean()

# 4. Resumen de Calificaciones
calificacion_promedio = grouped_gender['Calificación'].mean()
calificacion_maxima = grouped_gender['Calificación'].max()

# 5. Tabla de Resumen
resumen_genero = pd.DataFrame({'Edad Promedio': edad_promedio,
                              'Calificación Promedio': calificacion_promedio,
                              'Calificación Máxima': calificacion_maxima})

# 6. Cantidad de estudiantes por género
cantidad_estudiantes = grouped_gender['Nombre'].count()
print("Cantidad de estudiantes por género:\n", cantidad_estudiantes)

# 7. Ordenar la tabla de resumen por calificación promedio descendente
resumen_genero = resumen_genero.sort_values(by='Calificación Promedio', ascending=False)
print("\nTabla de resumen ordenada:\n", resumen_genero)

# 8. Agregar columna de categoría de edad
df['Categoría Edad'] = ['Joven' if edad < 19 else 'Adulto' for edad in df['Edad']]
print("\nDataFrame con columna de categoría de edad:\n", df)
```
