## Generador de Texto con Gemini 1.5 y Streamlit

### Requisitos Previos
Obtener una API Key de Google AI Studio:

* Visita https://makersuite.google.com/app/apikey
* Crea una cuenta o inicia sesi√≥n
* Genera una nueva API key

### Paso 1: Configuraci√≥n del Entorno

#### Crear un nuevo directorio para el proyecto
```bash
mkdir gemini-streamlit-app
cd gemini-streamlit-app
```

#### Crear un entorno virtual (opcional pero recomendado)
```bash
python -m venv .venv
# En Windows
.venv\Scripts\activate
```

#### Instalar las dependencias necesarias
```bash
pip install streamlit google-generativeai
```

### Paso 2: Estructura del Proyecto

Crear la siguiente estructura de archivos:
```
üìÅ proyecto_gemimi
    üìÅ .streamlit
        üìÑ secrets.toml
    üìÑ app.py   
```

### Paso 3: Configurar la API Key

#### Crear el archivo secrets.toml
1. Crear el directorio `.streamlit` si no existe
2. Dentro del directorio, crear el archivo `secrets.toml`
3. A√±adir el siguiente contenido:
```toml
[GEMINI]
api_key = "TU_API_KEY_AQUI"
```

### Paso 4: Implementar la Aplicaci√≥n

#### Crear el archivo app.py
Copiar el siguiente c√≥digo en `app.py`:
```python
import streamlit as st
import google.generativeai as genai

# Configura la API Key de Google Generative AI
genai.configure(api_key=st.secrets.GEMINI.api_key)

# Selecciona el modelo
model = genai.GenerativeModel("gemini-1.5-flash")

# Crea la interfaz de usuario con Streamlit
st.title("Generador de Texto con Gemini 1.5")
user_input = st.text_input("Ingresa tu texto:")

# Genera el texto si se presiona el bot√≥n
if st.button("Generar"):
    if user_input:
        response = model.generate_content(user_input)
        st.write("Respuesta:", response.text)
    else:
        st.write("Por favor ingresa un texto.")
```

### Paso 5: Ejecutar la Aplicaci√≥n

1. Abrir una terminal en el directorio del proyecto
2. Ejecutar el siguiente comando:
```bash
streamlit run app.py
```
3. La aplicaci√≥n se abrir√° autom√°ticamente en tu navegador predeterminado

## Ejemplo Analizador de camelCase en C√≥digo Java

```python
import streamlit as st
import google.generativeai as genai

# Configura la API Key de Google Generative AI
genai.configure(api_key=st.secrets.GEMINI.api_key)

# Selecciona el modelo

# Crea la interfaz de usuario con Streamlit

# √Årea para subir archivo

def analyze_camel_case(java_code):
    prompt = f"""
    Analiza el siguiente c√≥digo Java y eval√∫a el uso correcto de camelCase. 
    Verifica:
    1. Variables en camelCase

    Por favor, proporciona:

    """
    
    try:
        response = model.generate_content(prompt)
        return response.text
    except Exception as e:
        return f"Error al analizar el c√≥digo: {str(e)}"

# Genera el an√°lisis cuando se sube un archivo

    # Lee el contenido del archivo

    # Muestra el c√≥digo original

    st.code(java_code, language='java')
    
    # Bot√≥n para realizar el an√°lisis

    st.write("Por favor, sube un archivo Java para analizarlo.")

# Agrega informaci√≥n adicional
st.markdown("""

- **Variables**: camelCase (ej: nombreVariable)
- **M√©todos**: camelCase (ej: calcularTotal)
```
